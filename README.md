# Task/Note WebApp

A basic Task/Note taking webapp with minimal functionallity using C# ASP.NET for the backend and Vite+react for frontend.
This is a learning Project so my main focus was on basics like CRUD and Controller based APIs in ASP.NET.

## Self-Host

Setup Docker and self hosting with nginx and Cloudflare under taskapp.s-ose.de. 

## AI Usage
I only used AI for some boilerplate code and the css/design since i did not want to focus on design for this project.
