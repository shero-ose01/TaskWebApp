# Task/Note WebApp

A basic Task/Note taking webapp with minimal functionallity using C# ASP.NET for the backend and Vite+react for frontend.
This is a learning Project so my main focus was on basics like CRUD and Controller based APIs in ASP.NET.

## Functionality

Simple Account System (no oauth2), Creating, Updating, Listing and Deleting Tasks that have a Title, Description, State, and Date. 

## Self-Host

Setup Docker and self hosting with nginx and Cloudflare under <a href="taskapp.s-ose.de">taskapp.s-ose.de</a>. Later i'll add a Homepage under s-ose.de
