# Task/Note WebApp

A basic Task/Note taking webapp using C# ASP.NET for the backend and Vite+React for frontend.
This is a learning Project so my main focus was on basics like CRUD and Controller based APIs in ASP.NET.

## Functionality

### Simple Account System: 
- Email/Password registration with email verification (Resend for the Smtp)
- JWT auth

### Task CRUD:
Creating, Updating, Listing and Deleting Tasks that have a Title, Description, State, and Date. 

## Self-Host

Setup Docker and self hosting with nginx and Cloudflare Tunnel under <a href="taskapp.s-ose.de">taskapp.s-ose.de</a>. Later I'll add a Homepage under s-ose.de

## Running locally

- copy .env.example and rename it to .env and fill in the values
- docker compose up -d
